### CMT · 评论二级回复（自动化建议落 `test/fn-comment-test.js`）

后端已支持，前端回复入口已于 M2.1b 落地（详情页评论「回复」按钮 + 自己回复可删）。自动化 CMT-A01~05 全绿；CMT-M02/M03 转为开发者工具人工回归项。

| ID | 用例 | 预期 |
|----|------|------|
| CMT-A01 | 发布二级回复 | createComment 带 parentId → code 0；行落库 parent_id 正确；**日记 comment_count 不变**（仅一级计数） |
| CMT-A02 | 回复随父评论返回 | getComments → 父评论的 `replies` 数组含该回复，按时间正序 |
