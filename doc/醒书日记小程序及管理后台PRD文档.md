# 醒书日记微信小程序及管理后台PRD文档

# 项目名称

醒书日记小程序及配套管理后台

# 一、产品定位与目标

## 1.1 产品定位

醒书日记是一款专注于日记记录、分享与互动的轻量化小程序，搭配高效管理后台，为用户提供便捷的日记编写、查询、检索、收藏、点赞、评论、转发服务，同时支持用户根据需求设置日记权限，实现个性化记录与精准分享；管理后台则为运营者提供全面的数据监控、用户管理与内容管控能力，保障产品合规有序运行。

## 1.2 产品目标

- 用户端：打造简单易用、体验流畅的日记记录与互动平台，满足用户记录生活、分享心情、收藏优质内容的核心需求，提升用户活跃度与留存率。

- 运营端：通过管理后台实现对用户、日记及互动数据的全方位管理，支持灵活的内容筛选与查询，降低运营成本，为产品优化提供数据支撑。

- 产品理念：通过会员体系搭建，引导用户通过知识付费，查询会员的优质日记内容，并参与与其他会员的互动。

# 二、目标用户

## 2.1 核心用户

有日记记录习惯、乐于分享生活片段或心情感悟的人群，核心需求是便捷记录、安全存储、优质互动，隐私保护与内容个性化。

## 2.2 次要用户

- 喜欢浏览他人生活分享、寻找情感共鸣的用户，核心需求是发现优质日记内容、参与互动交流。

- 有个性化记录需求（如私密日记、会员专属内容分享）的用户，核心需求是灵活设置内容权限，保障隐私安全。

## 2.3 运营用户

产品运营人员、内容审核人员，核心需求是高效管理用户与内容、监控互动数据、排查违规内容，保障产品正常运营。

# 三、核心功能需求

## 3.1 小程序核心功能

### 3.1.1 醒书广场模块

- 日记列表展示：展示所有符合用户权限的日记卡片，包含日记标题、作者、内容摘要、发布时间、标签、权限范围、点赞数、收藏数、评论数、转发数。

- 日记标签可由管理员在后台维护，包括如：积极主动、以终为始、要事第一、双赢思维、移情聆听、统合综效、知止而后有定、修身为本、格物致知、君子慎独、诚中形外、十目所视、有斐君子、克明峻德、日新又新、听讼无讼、正心修身、絜矩之道、生财有道、以义为利。

- 日记多条件查询：支持按标题/内容关键词、标签、时间范围、作者昵称四种条件组合查询，提升内容查找效率。

- 日记详情查看：展示日记完整信息（标题、作者、全文、发布时间、标签、权限），支持点赞、收藏、评论、转发操作，返回列表时保持上次浏览位置，同步刷新互动数据。

- 日记点赞/取消点赞：用户可对日记进行点赞，再次点击取消点赞，互动数据实时同步。

- 日记收藏/取消收藏：用户可收藏日记，收藏状态实时切换，收藏内容可在“我的收藏”页面查看。

- 日记评论与回复：支持一级评论（对日记本身）和二级回复（对评论的回复），用户可删除自己发布的评论/回复。

- 日记转发：支持生成本日记专属海报（含小程序二维码），便于用户分享至第三方平台。

- 新建日记：右下角悬浮笔形图标，点击进入编写页面，支持输入标题、内容、选择标签、设置权限（公众/会员/自己），提交后同步至日记列表。

### 3.1.2 我的收藏模块

- 收藏日记列表：默认展示当前用户收藏的所有日记，页面样式与醒书广场一致，包含完整日记卡片信息。

- 收藏管理：支持在列表页或日记详情页取消收藏，取消后实时从收藏列表中移除，同步更新互动数据。

- 筛选与查询：支持沿用醒书广场的查询条件（标题/内容、标签、时间、作者），快速筛选收藏中的目标日记。

### 3.1.3 我的日记模块

- 个人日记列表：默认展示当前用户发布的所有日记，页面样式与醒书广场一致，包含完整日记卡片信息。

- 日记操作：支持新增日记（跳转至新建页面）、编辑自己的日记（修改标题、内容、标签、权限）、删除自己的日记（删除后不可恢复，同步删除相关互动数据，后台为逻辑删除）。

- 筛选与查询：支持沿用醒书广场的查询条件，快速筛选个人发布的目标日记。

### 3.1.4 会员中心模块

- 用户身份展示：明确显示当前用户身份（未授权手机号游客/已授权手机号游客/会员）。

- 权益查询：展示当前身份对应的权限的权益（如可查看的日记权限、可使用的功能），会员额外展示专属权益。

- 互动统计：展示用户发布日记的总数量、总点赞数、总收藏数、总评论数、总转发数，直观呈现内容影响力。

- 会员管理：会员用户展示会员有效期、剩余天数；非会员用户提供会员购买入口（一年365元），支持跳转至支付页面。

- 权限管控：根据用户身份限制功能使用，未授权用户引导授权手机号，会员解锁会员权限日记查看权限。

### 3.1.5 通用功能

- 用户授权：引导未授权手机号的游客授权，授权后解锁完整功能。

- 数据同步：所有互动操作（点赞、收藏、评论、转发）、日记操作（新增、编辑、删除）的数据实时同步，确保小程序与管理后台数据一致。

- 隐私保护：严格按照日记权限展示内容，未授权用户、普通游客无法查看会员/私密权限的日记。

## 3.2 管理后台核心功能

- 用户管理：查看所有用户信息（用户ID、手机号、身份类型、会员有效期、注册时间），支持按身份、注册时间筛选用户。

- 日记管理：查看所有用户发布的日记信息（完整内容、作者、发布时间、标签、权限、互动数据），支持按标题、内容、标签、时间范围、作者、权限、互动数据等条件组合查询。

- 互动数据管理：查看每篇日记的详细互动数据（点赞列表、收藏列表、评论/回复列表、转发记录），支持查看单条互动的具体信息（如评论者、评论时间）。

- 内容管控：支持查看日记详情，对违规日记进行删除操作，删除后同步删除小程序端内容及相关互动数据。

- 数据统计：直观展示平台总用户数、会员数、日记总数、总互动数（点赞、收藏、评论、转发），支持按时间范围查看数据变化。

# 五、页面说明（逐页说明）

## 5.1 小程序页面说明

### 5.1.1 醒书广场 - 日记列表页

- 页面布局：顶部搜索/筛选栏 + 中间日记卡片列表 + 右下角悬浮新建图标（笔形）。

- 顶部区域：左侧显示“醒书广场”标题，右侧设置筛选按钮，点击弹出筛选弹窗；搜索框支持输入关键词（标题/内容），回车触发搜索。

- 日记卡片：统一卡片样式，自上而下展示：标题（加粗，最多2行，超出省略）、作者（小字，灰色）、内容摘要（最多3行，超出省略）、发布时间+标签（标签为圆角矩形，浅底色，可多个并排）、权限标识（公众/会员/自己，图标+文字提示）、底部互动栏（点赞数+收藏数+评论数+转发数，搭配对应图标）。

- 悬浮图标：固定在右下角，笔形图标，点击跳转至新建日记页，点击时有轻微震动反馈。

- 交互说明：列表支持下拉刷新、上拉加载更多；点击日记卡片跳转至详情页；筛选弹窗确认后，列表实时刷新并定位至顶部；未授权用户点击新建/点赞/收藏/评论/转发按钮，跳转至手机号授权页。

### 5.1.2 醒书广场 - 日记查询筛选弹窗

- 弹窗布局：弹出式弹窗（从底部上滑），包含4个筛选条件：关键词（标题/内容）、标签、时间范围、作者，底部设置“重置”和“确认”按钮。

- 筛选条件：

    - 关键词：输入框，支持模糊查询。

    - 标签：下拉选择框，支持多选，可展示平台所有标签。

    - 时间范围：日期选择器，支持选择开始时间和结束时间，默认无限制。

    - 作者：输入框，支持输入作者昵称模糊查询。

- 交互说明：点击“重置”，清空所有筛选条件；点击“确认”，关闭弹窗并刷新日记列表，展示符合条件的内容；筛选条件可单独选择或组合选择，无选择时默认展示所有符合用户权限的日记。

### 5.1.3 醒书广场 - 日记详情页

- 页面布局：顶部返回按钮 + 中间日记内容区域 + 底部互动栏 + 评论列表区域。

- 顶部区域：左侧返回按钮（点击返回列表页，保持上次浏览位置），右侧无其他操作；标题居中（加粗，换行展示）。

- 日记内容区域：自上而下展示：作者（小字，灰色）、发布时间+标签+权限标识（与列表页一致）、完整日记内容（换行展示，支持分段）。

- 底部互动栏：固定在内容下方，从左至右依次为：点赞按钮（图标+数量，点赞后图标变色）、收藏按钮（图标+数量，收藏后图标变色）、评论按钮（图标+数量）、转发按钮（图标+数量），点击对应按钮触发对应操作。

- 评论列表区域：互动栏下方，展示所有一级评论，每条评论包含：评论者昵称、评论内容、评论时间、回复按钮（点击弹出回复输入框）；点击回复按钮，显示二级回复列表，支持输入回复内容并提交，可删除自己的评论/回复。

- 交互说明：点赞/收藏后，实时更新数量和按钮状态；点击评论按钮，聚焦至评论列表并弹出评论输入框；点击转发按钮，跳转至海报生成页；返回列表页时，列表自动刷新互动数据，并定位至上次进入详情页的位置；未授权用户触发操作时，跳转至授权页。

### 5.1.4 醒书广场 - 新建日记页

- 页面布局：顶部标题+保存按钮 + 中间输入区域 + 底部权限/标签设置区域。

- 顶部区域：左侧返回按钮（点击取消新建，弹出确认弹窗），中间“新建日记”标题，右侧“保存”按钮（默认置灰，输入标题和内容后激活）。

- 输入区域：

    - 标题输入框：提示文字“请输入日记标题”，最多输入30字，超出提示。

    - 内容输入框：提示文字“请输入日记内容”，支持多行输入、分段，无字数限制，显示输入字数统计。

- 底部设置区域：

    - 标签设置：提示文字“请选择日记标签”，支持多选，可自定义新增标签（需审核，暂不展示给其他用户）。

    - 权限设置：提示文字“请设置日记权限”，单选按钮，三个选项：公众（所有人可见）、会员（仅会员可见）、自己（仅自己可见），默认选择“公众”。

- 交互说明：输入标题和内容后，“保存”按钮激活，点击保存，提交日记并跳转至日记列表页（我的日记列表），提示“发布成功”；未授权用户点击保存，跳转至授权页；取消新建时，若已输入内容，弹出“是否放弃编辑”确认弹窗，确认后返回上一页。

### 5.1.5 醒书广场 - 日记转发海报页

- 页面布局：顶部返回按钮 + 中间海报展示区域 + 底部保存/分享按钮。

- 海报展示区域：居中展示日记专属海报，海报包含：日记标题、作者昵称、日记内容摘要（1-2行）、小程序二维码（跳转至该日记详情页）、产品Logo。

- 底部按钮：左侧“保存图片”按钮（点击保存海报至手机相册，提示“保存成功”），右侧“分享”按钮（点击弹出分享选项，支持分享至微信好友、朋友圈等）。

- 交互说明：海报生成后自动展示，加载过程中显示加载动画；点击返回按钮，返回日记详情页；保存图片需获取相册权限，未授权时提示引导授权。

### 5.1.6 我的收藏 - 收藏日记列表页

- 页面布局：与醒书广场日记列表页完全一致（顶部搜索/筛选栏 + 中间日记卡片列表 + 右下角悬浮新建图标）。

- 核心差异：默认展示当前用户收藏的日记，筛选/查询条件仅作用于收藏列表；日记卡片权限标识仍显示原日记权限，不影响收藏查看（自己收藏的会员/私密日记，自己可查看）。

- 交互说明：与醒书广场列表页一致，额外支持：点击收藏按钮取消收藏，取消后实时从列表中移除，提示“已取消收藏”；取消收藏后，同步更新日记详情页及广场列表页的收藏数据。

### 5.1.7 我的日记 - 个人日记列表页

- 页面布局：与醒书广场日记列表页一致，额外在日记卡片右侧增加“编辑”“删除”按钮（仅自己可见）。

- 核心差异：默认展示当前用户发布的所有日记，筛选/查询条件仅作用于个人日记；日记卡片右侧编辑/删除按钮，点击编辑跳转至编辑日记页，点击删除弹出确认弹窗。

- 交互说明：与醒书广场列表页一致，额外支持：

    - 编辑：跳转至编辑页面，填充原有日记内容、标签、权限，修改后保存，提示“编辑成功”，同步更新所有相关页面数据。

    - 删除：确认删除后，日记及相关互动数据（点赞、收藏、评论、转发）全部删除，提示“删除成功”，列表实时刷新。

### 5.1.8 我的日记 - 编辑日记页

- 页面布局：与新建日记页完全一致，仅顶部标题改为“编辑日记”。

- 核心差异：输入区域、标签、权限均填充原有日记内容，支持修改所有字段；标签修改后，原标签自动取消，新标签生效；权限修改后，实时影响日记可见范围。

- 交互说明：与新建日记页一致，保存后跳转至个人日记列表页，提示“编辑成功”；取消编辑时，若有修改，弹出确认弹窗；未授权用户无法进入编辑页面，跳转至授权页。

### 5.1.9 会员中心 - 会员信息与权益页

- 页面布局：顶部用户身份展示区 + 中间权益展示区 + 底部操作/统计入口。

- 用户身份展示区：居中展示用户头像（默认头像，可后续扩展上传）、昵称、身份标签（未授权手机号游客/已授权手机号游客/会员）；会员额外展示会员有效期、剩余天数（如“会员有效期至：2027-02-24，剩余360天”）。

- 权益展示区：按身份展示对应权益，用卡片样式呈现，每条权益搭配图标+文字：

    - 未授权手机号游客：可查看公众权限日记；需授权手机号解锁编写、点赞、收藏、评论、转发功能。

    - 已授权手机号游客：可查看公众权限日记；可使用编写、点赞、收藏、评论、转发功能；不可查看会员权限日记。

    - 会员：可查看公众、会员权限日记；可使用所有功能；享受会员专属标识（日记卡片、个人主页）。

- 底部操作/统计入口：两个按钮，“互动统计”（跳转至统计页）、“开通/续费会员”（非会员显示“开通会员”，会员显示“续费会员”，跳转至购买页）。

- 交互说明：未授权用户点击“开通会员”“互动统计”，跳转至授权页；会员点击“续费会员”，跳转至购买页；点击身份标签，弹出身份说明弹窗。

### 5.1.10 会员中心 - 互动统计页

- 页面布局：顶部标题“我的互动统计” + 中间数据卡片区域 + 底部数据明细入口。

- 数据卡片区域：四个卡片，分别展示：发布日记总数、总点赞数、总收藏数、总评论数+总转发数，每个卡片包含图标、数据、单位，卡片样式统一，色彩区分。

- 底部明细入口：“查看详细数据”按钮，点击跳转至数据明细页（展示每篇日记的具体互动数据）。

- 交互说明：数据实时同步，支持下拉刷新更新数据；未授权用户无法进入，跳转至授权页。

### 5.1.11 会员中心 - 会员购买页

- 页面布局：顶部标题“开通会员” + 中间会员套餐区域 + 底部支付区域。

- 会员套餐区域：展示唯一套餐（一年365元），卡片样式，包含：套餐名称（年度会员）、价格（365元/年）、套餐权益（可查看会员权限日记、享受会员专属标识）、有效期说明（开通后1年有效）。

- 底部支付区域：“微信支付”按钮（居中，加粗），下方显示支付说明（“支付成功后，立即开通会员权益”）。

- 交互说明：点击支付按钮，调用微信支付接口；支付成功后，跳转至会员中心，提示“会员开通成功”，同步更新会员身份和权益；支付失败/取消支付，提示对应信息，返回购买页；未授权用户无法进入，跳转至授权页。

### 5.1.12 通用页面 - 手机号授权页

- 页面布局：居中展示授权提示文字 + 微信手机号授权按钮 + 底部说明文字。

- 提示文字：“请授权手机号，解锁更多功能”（加粗，居中）。

- 授权按钮：微信标准手机号授权按钮（“获取手机号”），居中展示，点击调用微信手机号授权接口。

- 底部说明文字：“授权仅用于身份验证，不会泄露您的隐私”（小字，灰色，居中）。

- 交互说明：授权成功后，跳转至之前触发操作的页面，更新用户身份为“已授权手机号游客”；授权失败/取消授权，提示“授权失败，无法使用该功能”，返回上一页。

## 5.2 管理后台页面说明

### 5.2.1 首页 - 数据概览页

- 页面布局：顶部标题“数据概览” + 中间核心指标卡片 + 底部数据图表区域。

- 核心指标卡片：展示平台关键数据，包含：总用户数、会员数、日记总数、总点赞数、总收藏数、总评论数、总转发数，每个指标卡片包含数值、同比/环比变化（可选）。

- 数据图表区域：展示核心数据变化趋势，包含：用户增长趋势图（按时间）、日记发布趋势图（按时间）、互动数据趋势图（按时间），支持切换时间范围（日/周/月/年），支持下载图表。

- 交互说明：数据每10分钟自动刷新一次，支持手动刷新；点击指标卡片，跳转至对应的数据详情页（如点击总用户数，跳转至用户管理列表页）；图表支持hover查看具体数值。

### 5.2.2 用户管理 - 用户列表页

- 页面布局：顶部搜索/筛选栏 + 中间用户列表 + 底部分页控件。

- 搜索/筛选栏：支持按用户ID、手机号、身份类型（未授权/已授权/会员）、注册时间范围筛选，设置“重置”“搜索”按钮；支持批量导出用户列表（Excel格式）。

- 用户列表：表格形式，列包含：用户ID、手机号、昵称、身份类型、会员有效期（非会员显示“无”）、注册时间、操作（查看详情按钮）。

- 分页控件：支持设置每页显示条数（10/20/50条），支持上一页、下一页、跳转至指定页码。

- 交互说明：点击“查看详情”，跳转至用户详情页；筛选条件确认后，列表实时刷新并重置分页；导出按钮点击后，生成并下载Excel文件。

### 5.2.3 用户管理 - 用户详情页

- 页面布局：顶部返回按钮 + 中间用户基础信息区域 + 底部用户相关数据区域。

- 用户基础信息区域：展示用户详细信息，包含：用户ID、手机号、昵称、头像（默认/上传）、身份类型、会员有效期、注册时间、最后活跃时间。

- 用户相关数据区域：分为两个标签页，“发布日记”（展示该用户发布的所有日记，表格形式，包含日记ID、标题、发布时间、权限、互动数据）、“互动记录”（展示该用户的所有点赞、收藏、评论、转发记录，按时间排序）。

- 交互说明：点击日记ID/标题，跳转至日记详情页；点击互动记录中的日记相关信息，跳转至对应日记详情页；点击返回按钮，返回用户列表页并保持筛选条件和分页位置。

### 5.2.4 日记管理 - 日记列表页

- 页面布局：顶部搜索/筛选栏 + 中间日记列表 + 底部分页控件。

- 搜索/筛选栏：支持多条件组合筛选，包含：日记ID、标题/内容关键词、标签、时间范围（发布时间）、作者昵称、权限类型（公众/会员/自己）、互动数据范围（点赞数/收藏数/评论数/转发数），设置“重置”“搜索”按钮；支持批量删除日记、批量导出日记列表。

- 日记列表：表格形式，列包含：日记ID、标题、作者昵称、发布时间、标签、权限类型、点赞数、收藏数、评论数、转发数、操作（查看详情、删除按钮）。

- 分页控件：与用户列表页一致，支持自定义每页条数和页码跳转。

- 交互说明：点击“查看详情”，跳转至日记详情页；点击“删除”，弹出确认弹窗，确认后删除日记及相关互动数据，提示“删除成功”；筛选条件确认后，列表实时刷新；批量操作需先勾选日记，再点击批量按钮。

### 5.2.5 日记管理 - 日记详情页

- 页面布局：顶部返回按钮 + 中间日记基础信息区域 + 底部互动数据区域。

- 日记基础信息区域：展示日记完整内容，包含：日记ID、标题（加粗）、作者昵称（可点击跳转至用户详情页）、发布时间、标签、权限类型、完整内容（分段展示）。

- 互动数据区域：分为四个标签页，“点赞记录”“收藏记录”“评论记录”“转发记录”，每个标签页以表格形式展示对应互动数据，包含：互动ID、用户ID、用户名、互动时间，支持筛选和导出。

- 交互说明：点击作者昵称，跳转至用户详情页；点击互动记录中的用户ID，跳转至对应用户详情页；点击返回按钮，返回日记列表页并保持筛选条件和分页位置；支持删除违规评论（评论记录页操作）。

### 5.2.6 互动数据管理 - 各数据页（点赞/收藏/评论/转发）

- 页面布局：统一布局，顶部搜索/筛选栏 + 中间数据列表 + 底部分页控件。

- 筛选条件：支持按日记ID、日记标题、用户ID、用户昵称、时间范围筛选，评论记录额外支持按评论内容筛选。

- 数据列表：表格形式，包含对应互动的核心信息（如评论记录包含：评论ID、日记ID、日记标题、用户ID、用户名、评论内容、评论时间、回复对象（二级回复）、操作（删除））。

- 交互说明：与日记列表页一致，支持筛选、分页、导出、单个/批量删除（评论记录支持）；点击日记ID/标题，跳转至日记详情页；点击用户ID/用户名，跳转至用户详情页。

# 六、UI 配色与视觉规范

## 6.1 整体视觉风格

整体采用简约、清新、文艺的风格，贴合日记记录的情感属性，避免复杂装饰，注重内容可读性和操作便捷性；小程序以轻量化为主，管理后台以简洁高效为主，保持视觉统一性。

## 6.2 小程序配色规范

### 6.2.1 主色调

- 主色：#4A90E2（淡蓝色），用于顶部导航、按钮、图标高亮、重点文字，传递清新、舒适的氛围，贴合日记的治愈感。

- 辅助色：

    - #FF6B6B（淡红色）：用于点赞按钮高亮、删除按钮、提示错误信息。

    - #FFD166（淡黄色）：用于收藏按钮高亮、标签底色、提示警告信息。

    - #06D6A0（淡绿色）：用于操作成功提示、会员标识、转发按钮高亮。

- 中性色：

    - #333333：用于标题、重点文字，保证可读性。

    - #666666：用于普通文字、作者昵称、时间，柔和不刺眼。

    - #999999：用于提示文字、次要信息，区分层级。

    - #F5F7FA：用于页面背景、卡片底色，提升整体清爽感。

    - #E5E5E5：用于分割线、边框，区分不同区域。

### 6.2.2 各模块配色细节

- 日记卡片：底色#FFFFFF，边框#E5E5E5，标题#333333，作者/时间#999999，内容摘要#666666，标签底色#F5F7FA、文字#4A90E2，权限标识：公众#06D6A0、会员#FFD166、自己#999999。

- 按钮：

    - 主要按钮（保存、支付）：底色#4A90E2，文字#FFFFFF，点击后底色加深。

    - 次要按钮（重置、取消）：底色#F5F7FA，文字#666666，边框#E5E5E5。

    - 图标按钮（点赞、收藏、评论、转发）：默认#999999，高亮/选中后对应辅助色。

- 会员中心：会员身份标签底色#FFD166、文字#333333，权益卡片底色#F5F7FA、边框#E5E5E5，数据卡片底色#FFFFFF、边框#E5E5E5，文字#333333。

- 弹窗：背景半透明#000000（透明度30%），弹窗底色#FFFFFF，标题#333333，内容#666666，按钮配色与上述一致。

## 6.3 管理后台配色规范

### 6.3.1 主色调

- 主色：#3578F6（深蓝色），用于左侧导航栏、顶部标题栏、按钮、重点数据，传递专业、高效的氛围。

- 辅助色：与小程序一致，用于提示信息、操作按钮（删除#FF6B6B、成功#06D6A0、警告#FFD166）。

- 中性色：

    - #1F2937：用于页面标题、表格标题、重点文字。

    - #4B5563：用于普通文字、表格内容。

    - #9CA3AF：用于提示文字、次要信息。

    - #F3F4F6：用于页面背景、表格交替行底色。

    - #E5E7EB：用于分割线、边框、表格边框。

### 6.3.2 各模块配色细节

- 左侧导航栏：底色#3578F6，选中项底色#2563EB，文字#FFFFFF，未选中项文字#E0E7FF。

- 顶部标题栏：底色#FFFFFF，文字#1F2937，边框#E5E7EB。

- 表格：底色#FFFFFF，边框#E5E7EB，表头底色#F3F4F6，文字#1F2937，表格内容文字#4B5563，交替行底色#F9FAFB，hover行底色#F3F4F6。

- 按钮：主要按钮（搜索、导出）底色#3578F6，文字#FFFFFF；次要按钮（重置、查看）底色#F3F4F6，文字#4B5563，边框#E5E7EB；删除按钮底色#FF6B6B，文字#FFFFFF。

- 数据卡片：底色#FFFFFF，边框#E5E7EB，标题#9CA3AF，数据#1F2937，同比/环比文字：增长#06D6A0、下降#FF6B6B。

## 6.4 视觉通用规范

- 圆角：所有卡片、按钮、输入框、弹窗均采用8px圆角，避免尖锐边角，提升柔和感。

- 阴影：卡片、弹窗添加轻微阴影（阴影值：0 2px 8px rgba(0,0,0,0.08)），增强层次感，不突兀。

- 图标：统一采用线性图标，线条粗细2px，颜色与对应文字/按钮颜色一致，风格简约统一；小程序图标尺寸24px，管理后台图标尺寸20px。

- 文字：

    - 小程序：标题（18px，加粗）、正文（16px）、次要文字（14px）、提示文字（12px），字体采用“微信官方字体”（sans-serif fallback）。

    - 管理后台：标题（20px，加粗）、表格标题（16px，加粗）、正文/表格内容（14px）、提示文字（12px），字体采用“系统默认字体”（sans-serif fallback）。

- 间距：统一采用8px、16px、24px作为基础间距，页面内间距均匀，避免拥挤或松散，提升整洁度。

# 七、交互与状态设计

## 7.1 通用交互规范

- 反馈机制：所有操作（点击、提交、删除、授权等）均需提供明确反馈，如按钮点击震动、操作成功/失败提示弹窗（3秒自动消失）、加载动画（操作耗时超过1秒时显示）。

- 跳转逻辑：返回按钮统一返回上一页，保持页面层级清晰；跳转至详情页、编辑页等，返回时保持原页面的筛选条件、分页位置、滚动位置，提升用户体验。

- 手势操作：支持下拉刷新（列表页）、上拉加载更多（列表页）、左右滑动切换标签页（管理后台、会员中心）、点击空白处关闭弹窗（筛选弹窗、评论弹窗等）。

- 输入交互：输入框获取焦点时，边框高亮（主色）；输入错误/超出限制时，提示文字变红并显示具体错误信息；输入完成后，点击回车/确认按钮触发提交。

## 7.2 小程序核心交互设计

### 7.2.1 日记相关交互

- 新建/编辑日记：输入标题和内容后，保存按钮激活；未输入时，点击保存无反应并提示“请输入标题和内容”；保存成功后，提示并跳转至对应列表页；取消编辑时，有内容则弹出确认弹窗。

- 日记删除：点击删除按钮，弹出确认弹窗（“确定删除该日记吗？删除后不可恢复”），确认后删除并提示，列表实时刷新；删除失败时，提示“删除失败，请重试”。

- 日记查看：点击日记卡片跳转至详情页，返回时列表保持上次浏览位置，并自动刷新互动数据（点赞、收藏等）；详情页支持上下滑动查看内容和评论。

- 日记搜索/筛选：输入关键词/选择筛选条件后，点击确认，列表实时刷新并定位至顶部；无符合条件的内容时，显示“暂无相关日记”提示文字（居中，灰色，搭配空状态图标）。

### 7.2.2 互动相关交互

- 点赞/取消点赞：点击点赞按钮，图标立即变色，数量实时+1/-1，无其他弹窗（避免干扰）；未授权用户点击，跳转至授权页。

- 收藏/取消收藏：点击收藏按钮，图标立即变色，数量实时+1/-1，收藏时弹出短暂提示“已收藏”，取消时提示“已取消收藏”；未授权用户点击，跳转至授权页。

- 评论/回复：点击评论按钮，弹出评论输入框并聚焦，输入内容后点击发送，评论实时添加至列表顶部，数量+1；点击回复按钮，显示二级回复列表并弹出回复输入框，回复内容添加至对应评论下方；删除自己的评论/回复，点击删除按钮，弹出确认弹窗，确认后删除，数量实时-1；未授权用户点击，跳转至授权页。

- 转发：点击转发按钮，跳转至海报生成页，加载过程中显示加载动画；海报生成后，支持保存图片和分享，保存成功提示“图片已保存至相册”，分享后返回详情页。

### 7.2.3 会员相关交互

- 会员购买：点击开通/续费会员，跳转至购买页，点击支付按钮，调用微信支付接口；支付成功后，跳转至会员中心，提示“会员开通/续费成功”，同步更新会员身份和有效期；支付失败/取消，提示“支付失败，请重试”/“已取消支付”，返回购买页。

- 权限管控：未授权用户点击需授权的功能，跳转至授权页，授权成功后返回原页面并触发对应操作；普通游客点击会员权限日记，弹出提示“需开通会员查看该日记”，并提供开通会员入口；会员查看会员权限日记，无额外提示，正常展示。

## 7.3 管理后台核心交互设计

- 数据筛选/搜索：选择筛选条件/输入搜索关键词后，点击搜索，列表实时刷新并重置分页；无符合条件的数据时，显示“暂无相关数据”提示；筛选条件支持记忆，返回页面时保留上次筛选条件。

- 数据导出：点击导出按钮，弹出提示“正在生成文件，请稍候”，生成完成后自动下载Excel文件；导出失败时，提示“导出失败，请重试”。

- 内容删除：删除日记、评论等内容时，弹出确认弹窗，确认后删除并提示“删除成功”，列表实时刷新；批量删除时，需先勾选内容，再点击批量删除按钮，确认后批量删除。

- 分页交互：点击上一页/下一页，跳转至对应页码，列表平滑刷新；输入页码并回车，跳转至指定页码；每页条数切换后，列表重置为第一页并刷新。

- 数据刷新：管理后台数据每10分钟自动刷新一次，支持手动点击刷新按钮刷新；刷新过程中，数据区域显示加载动画，不影响其他操作。

## 7.4 状态设计

### 7.4.1 小程序状态设计

- 页面状态：正常状态：页面加载完成，所有元素（列表、按钮、输入框等）正常显示，可正常操作，数据完整展示。

- 加载状态：页面初始化、下拉刷新、上拉加载、筛选搜索、海报生成时，显示轻量加载动画（如圆形旋转图标），加载区域占位，避免页面抖动；加载耗时超过3秒时，添加“加载中，请稍候”文字提示。

- 空状态：列表页（广场、我的收藏、我的日记）无符合条件内容、互动记录为空时，显示空状态图标+提示文字（如“暂无相关日记”“暂无收藏内容”“暂无互动记录”），提示文字为灰色，居中展示，下方可添加引导按钮（如“去新建日记”“去广场发现”）。

- 异常状态：网络异常、接口请求失败时，显示异常图标+提示文字（“网络不佳，请检查网络后重试”“请求失败，请刷新重试”），居中展示，添加“刷新”按钮，点击可重新发起请求；无权限访问（如普通游客查看会员日记）时，显示权限不足提示+开通会员入口/返回按钮。

- 操作状态：
            未选中/未操作状态：按钮、图标（点赞、收藏等）显示默认颜色，输入框未获取焦点，弹窗未弹出，所有元素处于可触发状态。

- 选中/操作中状态：按钮点击时底色加深、图标轻微放大（反馈效果）；输入框获取焦点时边框高亮（主色），显示光标；点赞/收藏选中后图标显示对应辅助色，数量实时更新；筛选弹窗弹出时，遮罩层显示，弹窗内选中项高亮。

- 禁用状态：未授权用户的新建、点赞、收藏等按钮，未输入标题内容的保存按钮，处于禁用状态（置灰、无点击反馈），鼠标悬浮时可显示禁用提示（如“请授权手机号解锁功能”“请输入标题和内容”）。

- 操作成功/失败状态：操作完成后弹出对应提示弹窗（3秒自动消失），成功提示为绿色图标+“操作成功”（如“发布成功”“已取消收藏”），失败提示为红色图标+具体失败原因（如“删除失败，请重试”“保存失败，网络异常”），不干扰用户后续操作。

- 用户身份状态：
            未授权手机号游客：会员中心显示“未授权手机号游客”身份标签，点击需授权功能时，跳转至手机号授权页，权限标识仅显示“公众”类内容。

- 已授权手机号游客：会员中心显示“已授权手机号游客”身份标签，可正常使用基础功能，无法查看会员权限日记，会员购买入口显示“开通会员”。

- 会员用户：会员中心显示“会员”身份标签+会员有效期，解锁所有功能，可查看会员权限日记，日记卡片、个人身份标签显示会员专属标识，会员购买入口显示“续费会员”。

- 日记权限状态：
           公众权限：日记卡片显示“公众”标识（绿色图标+文字），所有用户均可查看、互动。

- 会员权限：日记卡片显示“会员”标识（黄色图标+文字），仅会员用户可查看，普通游客点击时提示权限不足并引导开通会员。

- 自己权限：日记卡片显示“私密”标识（灰色图标+文字），仅日记作者可查看、编辑、删除，其他用户无法在列表中看到该日记。

### 7.4.2 管理后台状态设计

- 页面状态：
            正常状态：页面加载完成，表格、图表、按钮、筛选栏等元素正常显示，数据完整渲染，可正常进行筛选、导出、删除等操作。

- 加载状态：页面初始化、数据筛选、搜索、导出、详情加载时，显示加载动画（表格区域占位加载、按钮加载状态），表格区域可显示“加载中...”文字提示；大数据量加载时，添加进度提示（如“加载中，已完成30%”）。

- 空状态：用户列表、日记列表、互动数据列表等无符合条件数据时，表格区域显示“暂无相关数据”提示文字（居中、灰色），无多余装饰，筛选栏保持可操作状态。

- 异常状态：网络异常、接口请求失败、权限不足时，显示异常提示（“网络异常，请检查网络后重试”“请求失败，请刷新页面”“您无此操作权限”），搭配对应图标，表格区域显示异常占位，添加“刷新”按钮（网络/请求异常时）或“返回上一页”按钮（权限不足时）。

- 分页状态：当前页码高亮显示，首页时“上一页”按钮禁用（置灰），末页时“下一页”按钮禁用（置灰）；无数据时，分页控件隐藏。

- 操作状态：
            未操作/未选中状态：按钮、筛选框、复选框等处于默认状态，复选框未勾选，按钮可点击，表格行无hover效果。

- 操作中状态：按钮点击后显示加载状态（按钮内旋转图标+文字“处理中...”），不可重复点击；筛选条件选择时，下拉框展开、复选框勾选状态实时切换；表格行hover时，底色变为浅灰色（贴合视觉规范），便于区分当前操作行。

- 禁用状态：无权限操作的按钮（如普通运营者的批量删除按钮）、未勾选数据时的批量操作按钮，处于禁用状态（置灰、无点击反馈），鼠标悬浮时显示禁用提示（如“无此操作权限”“请选择需操作的数据”）。

- 操作成功/失败状态：删除、导出、批量操作等完成后，页面顶部弹出提示条（左对齐，3秒自动消失），成功提示为绿色+“操作成功”（如“删除成功”“导出成功，文件已下载”），失败提示为红色+具体失败原因（如“删除失败，请重试”“导出失败，数据量过大”）；重要操作（如批量删除日记）失败时，可弹出确认弹窗提示是否重新操作。

- 数据状态：
            正常数据状态：表格、图表数据完整渲染，数值清晰，标签、权限等标识准确显示（与小程序端一致），图表hover时显示具体数据详情。

- 异常数据状态：数据缺失（如用户手机号为空、日记内容为空）时，表格对应单元格显示“无”（灰色）；数据异常（如互动数为负数）时，对应单元格显示红色数值，便于运营者排查。

- 筛选状态：筛选条件选择后，筛选栏显示已选条件（可单个删除），表格数据实时刷新，筛选状态持续保留，直至点击“重置”按钮清空。
