## v1.1.1 — 2026-07-11

日记格式工具条体验优化。

- 进入编辑时工具条**默认展开**（原默认缩小为右侧把手），点把手仍可缩起/展开。
- 键盘弹出时工具条**自动贴住键盘顶端**（含候选词栏引起的高度变化）；仍可手动拖走，下次弹键盘自动归位。

## v1.1.0 — 2026-07-11

日记富文本编辑、会员制写作与多项体验优化。

**日记富文本（本版核心）**
- 正文支持**五种颜色**（黑/深红/黄/蓝/绿）与**粗体、斜体、下划线**，以及**有序列表、无序列表、居中**排版。
- 编辑时浮动**格式工具条**：默认缩为右侧小把手不挡内容，点击展开 11 键单行工具条；可点住把手在全屏范围上下拖动。
- 详情页所见即所得展示带格式正文；旧日记与会员墙自动回退纯文本，摘要/搜索/海报不受影响。

**会员与登录**
- **写日记/编辑改为会员专享**：非会员点写日记弹窗引导开通会员（后端同步兜底防绕过）。
- 授权登录弹窗支持**填写昵称（可一键取微信昵称）与选择头像**（选填，默认微信图标）。
- 登录弹窗《用户服务协议》《隐私协议》可点击打开阅读；「服务协议」全站统一改名「用户服务协议」。

**互动体验**
- 列表**点赞/收藏秒响应**（乐观更新，失败自动回退）；修复「我的日记」列表点赞/收藏无反应。
- 可见范围改名：**公众可读 / 会员可读 / 个人草稿**，草稿换折角文档图标（卡片/详情同步）。
- 5 个一级页标题旁新增**客服入口**，点击联系微信客服（需配置客服链接后生效）。

**修复**
- 日记时间晚 8 小时（数据库时区根治，历史数据已校正）。
- 评论时间显示原始时间串 → 改为「刚刚/x分钟前/昨天 HH:MM」相对时间。
- 带格式日记再编辑变纯文本、会员被误判为非会员。

> 运营/部署侧：需重新部署云函数 createDiary / updateDiary / getDiaryDetail / getDiaryList / admin（富文本双字段）；正式环境数据库需执行 content_rich 加列；客服入口需在代码中填入企业微信客服链接（kfc…）。

## v1.0.4 — 2026-07-09

会员中心设置入口与合规能力。

- 会员卡右上角新增**设置入口（⚙）**，点击可查看《用户协议》《隐私协议》，并退出登录（原「设置/退出」小节收入此处）。
- 新增**用户协议、隐私协议查看页**。
- 新增**隐私授权处理**：选头像/图片、保存海报等敏感操作前按协议提示同意（需配合微信后台《用户隐私保护指引》生效）。
- 会员中心布局微调：会员卡与有效期卡固定，其余内容随页滚动。

> 运营侧：本版起小程序按版本自动连不同云环境（正式版→生产、体验/开发版→测试）；管理后台支持 dev/prd 双端口（`npm run dev` / `npm run dev:prd`）。

## v1.0.3 — 2026-07-09

会员中心与活动报名体验优化。

**会员中心**
- 页面改版：顶部身份卡、底部「醒书咨询」固定，中间内容可滚动。
- 互动统计准确显示（已记日记 / 获赞 / 被收藏 / 评论与转发）。
- 续费与开通会员的文案区分；去掉与昵称重复的「微信授权名」。

**活动报名**
- 报名弹窗自动带出你的姓名与联系方式，只读一键确认，无需每次输入。
- 首次报名若缺姓名或联系电话，先引导完善并同步到个人资料。

**界面统一**
- 醒书活动标题字号与首页一致；我的收藏 / 我的日记改为「标题+搜索固定、仅列表滚动」。

## v1.0.2 — 2026-07-08 · 首个体验版

首个对外体验版，醒书日记小程序 + 管理后台核心能力上线。

**小程序**
- 五个主页面：醒书广场、醒书活动、我的收藏、我的日记、会员中心；及日记详情、写日记、活动详情。
- 三态身份（游客 / 已授权 / 会员）与三种日记权限（公众 / 会员 / 私密），会员专属内容 30% 会员墙。
- 微信登录半屏弹窗；点赞 / 收藏 / 评论与二级回复；带参小程序码分享、转发好友、推荐人归属。
- 会员身份综合"身份 + 有效期"判断（过期即按非会员）；用户性别字段；广场列表下拉刷新。

**管理后台（Vue3）**
- 数据概览、用户管理、日记管理、互动数据、活动管理、会员订单管理。
- 会员订单线下开通/续期（含有效期字段）；用户可编辑会员身份、有效期、性别；登录超时自动提示。

**技术**
- 腾讯 CloudBase 云函数（25 个）+ MySQL（cpolar 隧道）；`npm test` 全量回归。
