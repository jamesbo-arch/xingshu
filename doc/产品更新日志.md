# 醒书日记 · 产品更新日志

> 面向产品/运营的版本更新记录，每个版本对应一个 git tag 与一次微信后台上传。
> 详细技术记录见 `doc/devlog.md`。最新版本在最上方。

## v1.0.4 — 2026-07-09

会员中心设置入口与合规能力。

- 会员卡右上角新增**设置入口（⚙）**，点击可查看《用户协议》《隐私协议》，并退出登录（原「设置/退出」小节收入此处）。
- 新增**用户协议、隐私协议查看页**。
- 新增**隐私授权处理**：选头像/图片、保存海报等敏感操作前按协议提示同意（需配合微信后台《用户隐私保护指引》生效）。
- 会员中心布局微调：会员卡与有效期卡固定，其余内容随页滚动。

> 运营侧：本版起小程序按版本自动连不同云环境（正式版→生产、体验/开发版→测试）；管理后台支持 dev/prd 双端口（`npm run dev` / `npm run dev:prd`）。

## v1.0.3 — 2026-07-09

会员中心与活动报名体验优化。

**会员中心**
- 页面改版：顶部身份卡、底部「醒书咨询」固定，中间内容可滚动。
- 互动统计准确显示（已记日记 / 获赞 / 被收藏 / 评论与转发）。
- 续费与开通会员的文案区分；去掉与昵称重复的「微信授权名」。

**活动报名**
- 报名弹窗自动带出你的姓名与联系方式，只读一键确认，无需每次输入。
- 首次报名若缺姓名或联系电话，先引导完善并同步到个人资料。

**界面统一**
- 醒书活动标题字号与首页一致；我的收藏 / 我的日记改为「标题+搜索固定、仅列表滚动」。

## v1.0.2 — 2026-07-08 · 首个体验版

首个对外体验版，醒书日记小程序 + 管理后台核心能力上线。

**小程序**
- 五个主页面：醒书广场、醒书活动、我的收藏、我的日记、会员中心；及日记详情、写日记、活动详情。
- 三态身份（游客 / 已授权 / 会员）与三种日记权限（公众 / 会员 / 私密），会员专属内容 30% 会员墙。
- 微信登录半屏弹窗；点赞 / 收藏 / 评论与二级回复；带参小程序码分享、转发好友、推荐人归属。
- 会员身份综合"身份 + 有效期"判断（过期即按非会员）；用户性别字段；广场列表下拉刷新。

**管理后台（Vue3）**
- 数据概览、用户管理、日记管理、互动数据、活动管理、会员订单管理。
- 会员订单线下开通/续期（含有效期字段）；用户可编辑会员身份、有效期、性别；登录超时自动提示。

**技术**
- 腾讯 CloudBase 云函数（25 个）+ MySQL（cpolar 隧道）；`npm test` 全量回归。
